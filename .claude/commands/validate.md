## HTML VALIDATION

### Level 1: Structure Check
```bash
# Verify HTML structure
npx html-validate index.html 2>/dev/null || echo "html-validate not available, manual check"
```

**Manual checks:**
- [ ] Single `<h1>` tag
- [ ] Heading hierarchy respected (no h3 before h2)
- [ ] All images have `alt` attributes
- [ ] All forms have labels
- [ ] `lang="fr"` on `<html>`
- [ ] Viewport meta tag present
- [ ] Charset meta tag present

---

## CSS VALIDATION

### Level 2: Style Check

**Manual checks:**
- [ ] All CSS variables defined in `:root`
- [ ] No `!important` abuse (max 5 occurrences)
- [ ] Responsive: media queries for mobile/tablet/desktop
- [ ] Animations use `transform`/`opacity` (not `width`/`height`/`top`/`left`)
- [ ] No inline styles in HTML (except generated by JS)

---

## JAVASCRIPT VALIDATION

### Level 3: JS Check
```bash
# Syntax check all JS modules
for f in modules/*.js; do node --check "$f" && echo "OK: $f" || echo "FAIL: $f"; done
```

**Manual checks:**
- [ ] No `var` usage (use `const`/`let`)
- [ ] No global variable pollution (except `window.stellarEngine`)
- [ ] All event listeners have corresponding cleanup
- [ ] IntersectionObserver properly disconnected
- [ ] Canvas animations use requestAnimationFrame

---

## PERFORMANCE VALIDATION

### Level 4: Performance Check

```bash
# Check image sizes
find ressources/ -type f \( -name "*.png" -o -name "*.jpg" -o -name "*.jpeg" \) -size +500k -exec ls -lh {} \;
```

**Manual checks:**
- [ ] No image > 500KB without srcset
- [ ] Preload/preconnect for critical resources
- [ ] CSS loaded before JS
- [ ] Fonts use `display=swap`
- [ ] Lazy loading on below-fold images

---

## SEO & A11Y VALIDATION

### Level 5: SEO & Accessibility

**SEO checks:**
- [ ] `<title>` tag present and descriptive
- [ ] `<meta name="description">` present
- [ ] Open Graph tags (og:title, og:description, og:image, og:url)
- [ ] Twitter Card tags
- [ ] Canonical URL

**A11Y checks:**
- [ ] Skip-to-content link
- [ ] ARIA labels on interactive elements
- [ ] Focus visible on all focusable elements
- [ ] Color contrast ratio >= 4.5:1
- [ ] Keyboard navigable (Tab, Enter, Escape)
